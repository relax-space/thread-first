## 线程安全或不安全的示例

共享变量: 多个线程之间都可以访问的变量

以下操作线程安全:

 1. 只有读 [查看例子](#1.只有读)
 2. 只有写 [查看例子](#2.只有写)
 3. 对共享变量的所有操作(读或写)都在一个锁内 [查看例子](#3.读写安全)

线程不安全的案例:

 4. 对共享变量同时做了读或写 [查看例子](#4.读写不安全)
 5. 对共享变量同时做了读或写,但是只对写加了锁 [查看例子](#5.读写不安全2)


### 1.只有读

多线程`安全`例子: 只对共享变量做读操作

``` python
from queue import Queue
from threading import Thread
from typing import List


def req(res_value: Queue, loop_count_list: List, i: int):
    total = 0
    for i in range(loop_count_list[i]):
        total += 1
    res_value.put(total)


def main(thread_count: int, loop_count_list: List):
    res_value = Queue()
    tasks = [Thread(target=req, args=(res_value, loop_count_list, i))
             for i in range(thread_count)]
    for i in tasks:
        i.start()
    for i in tasks:
        i.join()
    count = 0
    for i in range(res_value.qsize()):
        count += res_value.get()
    return count


if __name__ == '__main__':
    thread_count = 20
    loop_count = 100
    loop_count_list = [loop_count]*thread_count
    res = main(thread_count, loop_count_list)
    print(f'期待:{thread_count*loop_count},实际:{res}')
```


``` python
输出:
    期待:2000,实际:2000
```

### 2.只有写

多线程`安全`例子: 只对共享变量做写操作

``` python
from threading import Thread
from typing import List


def req(list: List, loop_count: int):
    total = 0
    for i in range(loop_count):
        total += 1
    list.append(total)


def main(thread_count: int, loop_count: int):
    list = []
    tasks = [Thread(target=req, args=(list, loop_count))
             for i in range(thread_count)]
    for i in tasks:
        i.start()
    for i in tasks:
        i.join()
    count = 0
    for i in list:
        count += i
    return count


if __name__ == '__main__':
    thread_count = 20
    loop_count = 1000
    res = main(thread_count, loop_count)
    print(f'期待:{thread_count*loop_count},实际:{res}')
```

``` python
输出:
    期待:20000,实际:20000
```

### 3.读写安全

多线程`安全`例子: 对共享变量的读写加锁

分析: 加锁之后,在get执行之后,put执行之前,不会切换出线程, 保证将要存入的变量(b+1),是从内存获取的最新数据

``` python
import time
from threading import Lock, Thread, current_thread
from typing import List

step_list: List = []


def sleep():
    for j in range(100000):
        j += 1


def req(lock: Lock, list: List):
    for i in range(0, 10):
        with lock:
            b = list[0]
            step_list.append(
                f'{"%05d" % current_thread().ident}-{"%02d"%i} get {b}')
            time.sleep(0.01)
            step_list.append(
                f'{"%05d" % current_thread().ident}-{"%02d"%i} put {b+1}')
            list[0] = b + 1


def main():
    list = [0]
    tasks = []
    lock = Lock()
    for i in range(2):
        tasks.append(Thread(target=req, args=(lock, list,)))
    for i in tasks:
        i.start()
    for i in tasks:
        i.join()
    return list[0]


if __name__ == '__main__':
    res = main()
    print(f'main期待:20,实际:{res}')
```

### 4.读写不安全

多线程`不安全`例子: 在多线程中，如果不加锁, 既做读又做写，会导致数据不对，举例说明不对的原因

分析: 在get执行之后,put执行之前,切换出了线程, 导致将要存入的变量(b+1),不是从内存获取的最新数据

关键步骤: [13220-06 put 7] 这一步的7是通过代码中b+1获取的,这b不是从内存中获取的值,而是这一步获取的值为6[17528-00 get 6]. 内存中b已经被另外的线程变成9了[17528-02 put 9],但是这一步b却是6,导致b+1的结果计算错误为7,正确应该是10

``` python
from threading import Thread, current_thread
from typing import List

step_list: List = []


def sleep():
    for j in range(100000):
        j += 1


def req(list: List):
    for i in range(0, 10):
        b = list[0]
        step_list.append(
            f'{"%05d" % current_thread().ident}-{"%02d"%i} get {b}')
        sleep()
        step_list.append(
            f'{"%05d" % current_thread().ident}-{"%02d"%i} put {b+1}')
        list[0] = b + 1


def main():
    list = [0]
    tasks = []
    for i in range(2):
        tasks.append(Thread(target=req, args=(list,)))
    for i in tasks:
        i.start()
    for i in tasks:
        i.join()
    return list[0]


if __name__ == '__main__':
    res = main()
    print(f'main期待:20,实际:{res}')
    for i in step_list:
        print(i)
```

### 5.读写不安全2

多线程`不安全`例子: 在多线程中，既做读又做写，如果`只对写加锁`,也会导致数据不对

``` python
from threading import Lock, Thread, current_thread
from typing import List

step_list: List = []


def sleep():
    for j in range(100000):
        j += 1


def req(lock: Lock, list: List):
    for i in range(0, 10):
        b = list[0]
        step_list.append(
            f'{"%05d" % current_thread().ident}-{"%02d"%i} get {b}')
        sleep()
        step_list.append(
            f'{"%05d" % current_thread().ident}-{"%02d"%i} put {b+1}')
        # 只锁写
        with lock:
            list[0] = b + 1


def main():
    list = [0]
    tasks = []
    lock = Lock()
    for i in range(2):
        tasks.append(Thread(target=req, args=(lock, list,)))
    for i in tasks:
        i.start()
    for i in tasks:
        i.join()
    return list[0]


if __name__ == '__main__':
    res = main()
    print(f'main期待:20,实际:{res}')
    for i in step_list:
        print(i)
```
